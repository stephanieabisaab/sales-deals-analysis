# Sales deals analysis

A small but complete data project in two parts: Python fundamentals, then a real analysis of a sales pipeline with pandas.

## Part 1, fundamentals

`final_exam_part1_fundamentals.py`

Loops, functions and string handling through small classic exercises: printing squares, checking a list of grades, counting vowels in a quote about statistics.

## Part 2, a pandas analysis

`final_exam_part2_pandas_analysis.py`

I load a raw Excel export of sales deals, keep only the useful columns, rename them, fix the dates and the amounts, and drop the broken rows. Then I answer a few clear questions with pandas and matplotlib: how many deals per year, which sources bring the most deals, and how much money each one represents. The cleaned dataset is saved to a new Excel file at the end.

## How to run

Python 3 with pandas, openpyxl and matplotlib. Update the path to the Excel file at the top of part 2 before running.
