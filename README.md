# Python final exam

My final exam for the Python course at EDC Paris, in two parts.

## Part 1, fundamentals

Loops, functions and string handling through small classic exercises: printing squares, checking a list of grades, counting vowels in a quote about statistics.

## Part 2, a small data analysis in pandas

I load a raw Excel export of sales deals, keep only the useful columns, rename them, fix the dates and the amounts, and drop the broken rows. Then I answer a few simple questions with pandas and matplotlib: how many deals per year, which sources bring the most deals, and how much money each one represents. The cleaned dataset is saved to a new Excel file at the end.

The repo also contains a text visualisation notebook that belongs to another lab.

## How to run

Python 3 with pandas, openpyxl and matplotlib. Update the path to the Excel file at the top of part 2 before running.
